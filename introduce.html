3DMAX course plan: This course is divided into three topics. Comprehensive assessment will be conducted in the fourth week.

JavaScript course plan: This course will take you to learn the very important technology JavaScript in front-end development, which is the cornerstone of web page interaction with users. A complete JavaScript consists of the following three partsThe core of learning in the first week: ECMAScript, which is the learning goal of the first week. In the second week, learn the document object model: DOM; Week 3: learn the browser object model: BOM; Comprehensive assessment will be conducted in the fourth week.
